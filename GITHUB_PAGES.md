# 🚀 Deploy Lunar to GitHub Pages

## Quick Start (3 Steps)

### 1. Create GitHub Repository

**Option A - Using GitHub CLI (Fastest):**
```bash
gh repo create lunar --public --source=. --remote=origin --push
```

**Option B - Manual:**
```bash
# Initialize git (already done)
git add .
git commit -m "Initial commit"
git branch -M main

# Create repo on github.com/new, then:
git remote add origin https://github.com/YOUR_USERNAME/lunar.git
git push -u origin main
```

---

### 2. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** tab
3. Click **Pages** in left sidebar
4. Under "Build and deployment":
   - **Source**: `GitHub Actions` (not "Deploy from a branch")

That's it! GitHub Actions will automatically build and deploy your site.

---

### 3. Access Your Site

Your site will be live at:
```
https://YOUR_USERNAME.github.io/lunar/
```

Deployment takes ~1-2 minutes. Check the **Actions** tab to see progress.

---

## How It Works

The `.github/workflows/deploy.yml` file:
- ✅ Triggers on every push to `main` branch
- ✅ Builds your React + Vite app
- ✅ Deploys to GitHub Pages automatically
- ✅ Updates your site on every commit

---

## Custom Domain (Optional)

1. Go to Settings → Pages → Custom domain
2. Enter your domain (e.g., `lunar.ai`)
3. Add CNAME record in your DNS provider
4. GitHub handles SSL automatically

---

## Troubleshooting

**Site not updating?**
- Check Actions tab for build errors
- Make sure you pushed to `main` branch
- Hard refresh browser (Cmd+Shift+R)

**404 error?**
- Wait 1-2 minutes for deployment
- Check that Pages source is set to "GitHub Actions"

**Build fails?**
- Run `npm run build` locally to test
- Check Node version (should be 20+)

## Project Structure

```
lunar/
├── .github/workflows/deploy.yml  # Auto-deploy script
├── src/                          # React source code
├── public/                       # Static assets
├── dist/                         # Built files (NOT committed)
├── package.json                  # Dependencies
└── vite.config.ts               # Build config
```

**Note:** The `dist/` folder is generated by `npm run build` and is NOT committed to git. GitHub Actions builds it automatically.
